A Few Projects

Here's some things that I've recently worked on. Check them out!

Twitter Toxicity Sentiment Analysis

Built a NLP classifier with PyTorch and Fastai from a Kaggle training set to identify toxicity within online comments, and then deployed it using Starlette and the Twitter API to grab recent tweets to analyze. Check out my repo and presentation!

API Classifier on Voter Registration

Scraped Wikipedia for name data to train a model that classifies ethnicity by name, and then used the trained model to determine Asian American voter representation from voter registration records of local Ohio counties. Check out my slides!

In Response to Microaggressions

Wrote an opinion article for Hearhere, the journal for Christian thought for the Claremont Colleges, on my experiences with microaggressions as a Chinese-American growing up in the Midwest. Check it out here!
